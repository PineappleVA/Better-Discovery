Política de privacidad

Última actualización: 19 de junio de 2026.

Esta política describe qué datos puede tratar Better Discovery, por qué se usan y qué control tienes sobre ellos. El sitio utiliza servicios externos para autenticación, almacenamiento y distribución de contenido.

1. Datos que puedes compartir

Según cómo uses la app, puedes proporcionar email de acceso, nombre público, biografía, creaciones HTML, comentarios, autores seguidos y preferencias guardadas en el navegador.

2. Datos generados por el uso

El sitio puede registrar información técnica necesaria para funcionar, como sesiones activas, identificadores internos, preferencias locales y el estado de elementos visibles de la interfaz. También puede almacenar metadatos de las creaciones para mostrar autor, fecha, likes y comentarios.

3. Finalidad del tratamiento

  • Autenticar usuarios y mantener sesiones.
  • Publicar y mostrar creaciones HTML.
  • Guardar comentarios, likes y perfiles visibles.
  • Permitir seguimiento entre personas usuarias y calcular el espacio usado.
  • Mejorar la seguridad, moderación y experiencia de navegación.

4. Conservación

Conservamos el contenido mientras sea útil para prestar el servicio o hasta que se elimine según las reglas del proyecto o las capacidades de los sistemas externos usados. Parte de tu configuración se guarda localmente en tu navegador.

5. Cookies y almacenamiento local

Better Discovery puede usar almacenamiento local del navegador para recordar tu perfil público, a quién sigues, qué has marcado como propio y qué contenido has valorado. Puedes borrar estos datos desde la configuración de tu navegador.

6. Terceros

El sitio utiliza servicios de terceros como Supabase. Cada proveedor puede tratar datos de acuerdo con sus propias políticas y condiciones. Te recomendamos revisar la documentación de dichos servicios.

7. Tus derechos

Puedes pedir acceso, corrección o eliminación de tu contenido cuando el diseño del sistema lo permita. También puedes dejar de usar la cuenta, limpiar el almacenamiento local o contactar con el proyecto para plantear una revisión.

8. Seguridad

El HTML de las creaciones se muestra dentro de iframes con sandbox para reducir el impacto de contenido no confiable. Aun así, ningún sistema puede garantizar seguridad absoluta, por lo que debes publicar contenido responsable.

9. Menores

Si el proyecto se usa por menores de edad, debe hacerse bajo supervisión de una persona adulta cuando sea necesario y respetando las leyes aplicables en cada jurisdicción.

10. Cambios

Podemos actualizar esta política para reflejar cambios técnicos, legales o de producto. La versión publicada aquí será la referencia principal.

11. Discovery+

La suscripción Discovery+ almacena datos específicos de suscripción como el estado, la fecha de expiración y el plan asociado al correo del usuario. Estos datos se usan únicamente para habilitar beneficios dentro de la plataforma y no se comparten con terceros.
