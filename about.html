Acerca de Better Discovery

Better Discovery es una galería abierta y comunitaria para creaciones HTML. Sube tus páginas HTML, demos interactivas, juegos o herramientas y deja que cualquier persona las vea, juegue, reaccione y comente — sin necesidad de programar.

✨ Funciones

📤

Subir HTML

Pega hasta 1 MB de código HTML y publícalo al instante.

🖥️

Vista previa en vivo

Cada creación corre en un iframe con sandbox aislado.

❤️

Me gusta

Muestra apoyo a tus creaciones favoritas con un solo clic.

💬

Comentarios

Deja feedback, preguntas o apoyo a las personas creadoras.

👤

Perfil y creaciones

Edita tu perfil, sigue a otras personas y revisa tu espacio usado.

🔍

Búsqueda

Encuentra creaciones por título o descripción con búsqueda instantánea.

📱

Responsive

Funciona en escritorio, tablet y dispositivos móviles.

🔐 Acceso

Usa la pantalla de Login para entrar con tu cuenta y luego explora, sube, comenta y administra tu perfil.

📄 Páginas

El proyecto contiene varias páginas:

  • index.html — Explora y descubre todas las creaciones HTML
  • upload.html — Sube una nueva creación HTML (hasta 1 MB)
  • view.html?id=… — Visualiza, da like y comenta una creación
  • profile.html — Edita tu perfil, ve tus creaciones y seguidores
  • about.html — Esta página: resumen del proyecto
  • terms.html — Términos y condiciones
  • privacy.html — Política de privacidad

🛠️ Tech Stack

HTML + CSS + JavaScript puro (sin build tools ni frameworks) y Supabase para autenticación y persistencia. Todo está pensado para alojarse en GitHub Pages.

🔒 Seguridad

Todo el HTML subido se ejecuta dentro de un <iframe> con sandbox usando un atributo sandbox restringido.
